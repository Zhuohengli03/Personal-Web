# 网页背景资源

把背景图片或视频放在此文件夹中，站点会自动用作全页背景。

## 使用方式

**图片背景**（任选其一即可）：
- 放入 `background.jpg` 或 `background.png`
- 建议尺寸：1920×1080 或更大，比例 16:9 较佳

**视频背景**（任选其一即可）：
- 放入 `background.mp4` 和/或 `background.webm`
- 会自动静音、循环、铺满全屏播放

**优先级**：若同时有图片和视频，**视频会叠在图片上方**显示（即以视频为主）。  
若都不放任何文件，则使用默认深色背景。

## 建议

- 图片/视频建议偏暗或低对比，以保证前景文字清晰可读（页面会加一层半透明暗色遮罩）。
- 视频格式推荐 MP4（H.264），兼容性较好；可再加一份 WebM 以兼顾部分浏览器。
